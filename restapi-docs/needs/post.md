# Создать потребность

Создает новую потребность. Каждое создание потребности, даже если переданы одни и те же данные, создает новый экземпляр потребности.

**URL** : `/api/needs/`

**Method** : `POST`

**Auth required** : Нет

**Permissions required** : Нет

**Обязательные данные**

```json
{
    "user": <integer>
}
```

**Пример передаваемых данных** 

```json
{
    "user": 81007,
    "number1": 1,
    "number2": 9,
    "datecreate": 1597974855,
    "dateneed": 1601398800,
    "dateready": 1597975285,
    "datesend": 0,
    "draw": "",
    "import": "",
}
```

## Успешный запрос

**Code** : `201`

**Пример возвращаемых данных**

```json
{
    "id": 4288,
    "user": 81007,
    "number1": 1,
    "number2": 9,
    "datecreate": 1597974855,
    "dateneed": 1601398800,
    "dateready": 1597975285,
    "datesend": 0,
    "draw": "",
    "import": "",
    "createdAt": "2020-08-21T08:54:15.000Z",
    "updatedAt": "2020-08-21T08:54:15.000Z",
}
```

## Отвер при ошибке

**Condition** : Если произошла ошибка на сервере.

**Code** : `500`

**Content** : `{
        message: err.message || `Произошла ошибка при создании Needs.`,
      }`
